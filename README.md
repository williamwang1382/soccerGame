# soccerGame
# a game inspired by the popular Head Soccer mobile game!
# Requires JavaFX

# Controls:
# 1-Player Mode:
# W to jump
# A to move left
# D to move right
# space to quit the match
# 
# 2-Player Mode:
# 1st Player has the same controls as the 1-Player Mode
# 2nd Player controls:
# I to jump
# J to move left
# L to move right
